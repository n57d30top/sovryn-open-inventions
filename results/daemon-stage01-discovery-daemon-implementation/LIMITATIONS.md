# Limitations

- No Fund Gate has passed in this setup package.
- The daemon implementation provides silent search state, identity, graveyard, checkpoint, and Fund Gate mechanics; it does not itself validate a discovery candidate.
- Routine partial, weak, or promising-but-unvalidated candidates are intentionally not published as public results.
- The local search state is resumable but bounded by available runtime in each invocation.
- All external review, prize-significance, broad intelligence, adoption, legal, medical, wet-lab, unsafe, and universal claims are explicitly outside scope.
