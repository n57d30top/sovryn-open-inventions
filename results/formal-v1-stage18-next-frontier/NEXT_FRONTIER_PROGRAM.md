# Next Frontier Program

Selected next frontier: integrate_proof_assistant_route.

Rationale: richer generators alone still yielded counterexample-narrowed families. The next formal step should make proof constraints shape generation earlier, instead of generating first and trying to rescue weak families later.

Eight-week outline:

- Weeks 1-2: select a small proof-assistant-compatible subset of graph or recurrence families.
- Weeks 3-4: encode generator outputs as typed conjecture templates with lemmas.
- Weeks 5-6: run proof-guided counterexample search and reject templates with missing invariants.
- Weeks 7-8: freeze one bounded family only if proof obligations and replay artifacts are coherent.
