# Eight-Week Plan

1. Build a failure-derived anomaly wall from baseline-dominated cases.
2. Generate new candidates that explicitly beat metadata-count and sparsity baselines.
3. Reject candidates without executable falsifiers.
4. Freeze a smaller high-risk prediction set.
5. Select fresh public targets after freeze.
6. Run baseline-first execution.
7. Run replay and mutation stress early.
8. Publish a kill-or-survive report.
