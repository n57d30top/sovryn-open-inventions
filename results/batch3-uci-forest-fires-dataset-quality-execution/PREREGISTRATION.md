# Preregistration

Research question: Does a distribution-aware dataset audit surface quality and modeling caveats that a schema-only audit misses for UCI Forest Fires?

Metrics fixed before execution:
- row count
- column count
- missing cells
- duplicate rows
- zero-area rate
- maximum area
- positive-area average

Kill criteria:
- dataset unavailable
- zero-area rows hidden
- schema-only audit claimed sufficient
