# Target Selection

Selected targets include two high split-risk targets, one low-risk control, one ambiguous target, and targets with source/protocol files or useful controls.

- UCI HAR Smartphones: https://archive.ics.uci.edu/dataset/240/human+activity+recognition+using+smartphones
- UCI Statlog Shuttle: https://archive.ics.uci.edu/dataset/148/statlog+shuttle
- UCI Statlog Landsat Satellite: https://archive.ics.uci.edu/dataset/146/statlog+landsat+satellite
- UCI Vehicle Silhouettes: https://archive.ics.uci.edu/dataset/149/statlog+vehicle+silhouettes
- UCI Letter Recognition: https://archive.ics.uci.edu/dataset/59/letter+recognition
- UCI Optical Digits: https://archive.ics.uci.edu/dataset/80/optical+recognition+of+handwritten+digits
- UCI Image Segmentation: https://archive.ics.uci.edu/dataset/50/image+segmentation
- scikit-learn digits control: https://scikit-learn.org/stable/modules/generated/sklearn.datasets.load_digits.html
