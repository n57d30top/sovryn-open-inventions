# UCI Banknote Authentication Baseline-Dominance Challenge

A container-local benchmark run evaluated simple threshold baselines and a two-feature challenger on UCI Banknote Authentication; the challenger was rejected because the best variance threshold baseline achieved F1 0.855 versus challenger F1 0.716.

Final result: benchmark_challenge_negative_result.
