# Eight Week Plan

1. Build a preregistered target set for benchmark/protocol evidence-triad checks.
2. Freeze predictions across at least 80 targets.
3. Run stronger baselines: popularity, artifact count, documentation length, and domain family.
4. Run holdouts over fresh benchmark/protocol targets.
5. Replay package receipts independently.
6. Run a focused kill week before any readiness promotion.
7. Publish either a strengthened candidate package or a failure package.
8. Decide whether to continue the domain or return to broader search.
