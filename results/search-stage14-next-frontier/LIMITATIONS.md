# Limitations

- Only safe public computational, data, package-metadata, and formal-sequence targets were used.
- The rejected Receipt-Stable Descriptor Residual Candidate is not revived or rebranded as a discovery candidate.
- New candidates must pass baselines, counterexamples, holdouts, replay, and kill week before any stronger status.
- Network endpoints can change; replay caveats are documented where source stability is not guaranteed.
