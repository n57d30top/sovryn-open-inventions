# Death-Cause Rules

- Reject if simple baselines explain most candidate signal.
- Reject if counterexample density is high before promotion.
- Reject if decisive evidence cannot be replayed or has no replay plan.
- Downgrade if mutations do not break the candidate as expected.
- Reject if scope collapses to a heuristic-only fragment.
- Reject if a rival explanation has equal or better explanatory power.
- Downgrade if artifact explanation remains unresolved.
- Reject if holdout support is weak or absent.
- Reject if the signal is not actionable as a prediction generator.
- Reject if non-obvious predictions are mostly wrong or inconclusive.
