# Reproduce

1. Build the Product.
2. Run the Product test suite.
3. Run route status and audit commands.
4. Inspect .sovryn/route/status.json, target-universe.json, batch-results.json, route-scorecard.json, and public-packages.json.
5. Rebuild the public corpus site and run corpus audits.

The route pilot uses bounded public-safe descriptors and route evidence summaries.
