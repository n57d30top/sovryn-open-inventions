# Allowed Fund Labels

Allowed Fund labels are deliberately narrow:

- `externally_review_ready_candidate`
- `bounded_validated_conjecture_candidate`
- `checked_proof`
- `checked_refutation_with_high_external_value`
- `new_class_level_10x_candidate` only for the OS/x10 branch

The daemon does not accept partial or promising-but-unvalidated states as completion.
