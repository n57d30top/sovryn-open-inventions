# No-Notify Statuses

The following statuses are internal only and cannot trigger a user notification:

- `no_signal`
- `weak_signal`
- `partial_signal`
- `promising_but_unvalidated`
- `promising_with_strong_caveats`
- `killed_by_baseline`
- `killed_by_counterexample`
- `killed_by_replay`
- `killed_by_identity_drift`
- `killed_by_known_pattern`
- `killed_by_rival_theory`
- `candidate_graveyard_updated`
- `continue_searching`
