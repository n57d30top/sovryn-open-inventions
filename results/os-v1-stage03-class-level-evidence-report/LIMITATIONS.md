# Limitations

- This is a candidate package based on bounded route-policy-v3 workload evidence.
- Passing classes are class-level candidates inside the measured workload, not a claim about all tasks.
- Repo-package reproduction, formal counterexample, and temporal evaluation remain partial classes.
- Public packages are minimum-evidence packages, not expert review reports.
- The package has no outside-use evidence and no outside confirmation claim.
- Time-to-evidence estimates are benchmark artifacts and require continued calibration.
