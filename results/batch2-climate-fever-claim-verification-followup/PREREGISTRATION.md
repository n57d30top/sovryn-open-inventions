# Preregistration

Research question: Can public Climate-FEVER metadata be reconstructed into a reliable source-card schema before any model-level improvement is claimed?

Metrics fixed before evaluation:
- field recall
- schema reconstruction completeness
- reproduction score
- recorded unreproduced components

Kill criteria:
- no public source card available
- expected evidence fields cannot be identified
- any full benchmark win is claimed without model-level reproduction

Publication criteria: publish only if the result includes external target binding, concrete metrics, failures/losses, limitations, reproducibility instructions, and public hygiene pass.
