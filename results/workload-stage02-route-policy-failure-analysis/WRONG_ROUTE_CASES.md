# Wrong Route Cases

Twenty prior cases were reviewed as representative policy stress examples. Weakness classes found: mixed dataset/repo targets, doc-only package targets, benchmark protocols without runnable artifacts, temporal claims with generic dataset language, formal patterns without proof route, claim-review targets requiring safety review first, tool-claim targets over-routed to deep validation, not-testable decisions without second confirmation, unsafe near-misses routed too late, and package candidates with insufficient reproduce instructions.
