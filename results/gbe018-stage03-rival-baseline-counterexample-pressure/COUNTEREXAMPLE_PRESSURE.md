# Counterexample Pressure

Counterexample pressure did not fully refute the evidence-triad idea, but it generated enough narrowing pressure that a direct deep-validation launch would be premature.

- Benchmark/protocol targets with complete-looking metadata can still lack replayable evidence.
- Targets with weaker public metadata may still reproduce when execution paths are simple.
- Protocol popularity can coincide with both documentation and replayability.
- Holdout failures show the signal does not generalize cleanly under the current candidate definition.
