# Program Goal

Run a sustained evidence-driven research program over multiple batches using the capabilities already present.

The program must demonstrate continuity:

- Reuse prior results.
- Reuse or retire prior tools.
- Select targets based on prior evidence.
- Perform deeper studies.
- Attack older claims.
- Update knowledge and confidence.
- Select the next research direction based on accumulated evidence.

Primary success condition: Sovryn must demonstrate that it is not only producing isolated results, but learning across results.
