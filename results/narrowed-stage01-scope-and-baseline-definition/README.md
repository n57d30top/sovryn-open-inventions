# Narrowed Stage 01 Scope and Baseline Definition

The narrowed Receipt-Stable Descriptor Residual Candidate says that, inside safe public package metadata and sparse-table artifacts, a residual is only evidence-bearing when the source receipt is stable, the descriptor is explicitly computable, and the residual remains after metadata-count, sparsity, shuffled-descriptor, random-control, and popularity/proxy baselines. If stronger baselines fully explain the residual, or replay fails on the decisive receipt or descriptor, the candidate is downgraded or killed as a validation candidate.

## Scope
Bounded internal computational validation only. No outside-party review, uptake, prize-level claim, broad autonomous-intelligence claim, medical/legal/wet-lab capability, hazardous capability, or universal validity is claimed.
