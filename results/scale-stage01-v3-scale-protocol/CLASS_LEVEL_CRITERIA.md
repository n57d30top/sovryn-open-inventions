# Class-Level Criteria

A class passes only when it has multiple task-level 10x timing candidates, acceptable evidence completeness, package completeness, low route-error burden, and public packages with limitations.

| Threshold | Value |
| --- | --- |
| Minimum targets per class | 30 |
| Minimum passing classes for OS v1 candidate consideration | 4 |
| Overall route errors/caveats | <= 10% |
| Unresolved package quality issues | <= 5% |
| Public packages | >= 80 |
| Broad acceleration claim | forbidden |
