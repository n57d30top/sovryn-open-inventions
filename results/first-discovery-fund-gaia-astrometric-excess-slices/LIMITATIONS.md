# Limitations

- This is not external validation.
- This is not a Nobel, Einstein-level, breakthrough, or universal scientific claim.
- The claim is bounded to four public Gaia EDR3 RA slices and 160 rows selected by the recorded ADQL queries.
- The simple residual model is intentionally small; stronger Gaia-specific calibration, scan-law, crowding, source-quality, or selection-function rivals can explain the signal.
- Extended validation identifies RUWE as a catalog-quality rival that explains/collapses the primary cross-slice residual support after G/color/RUWE adjustment.
- Because the RUWE-adjusted primary residual magnitude is only `0.0263` and cross-slice support is `false`, this public package is not discovery-scored.
- One added north-declination holdout is counterexample-collapsed by single-slice dominance. The south-declination holdout is supportive only with RUWE caveats.
- The package shows raw replayability of the Product-bound scalar metrics, not independent expert acceptance.
- The Product package text includes inherited generic caveats. The public reviewer-facing interpretation here is narrower and should control external review.
