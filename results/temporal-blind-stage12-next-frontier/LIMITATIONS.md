# Limitations

- This is a bounded instrument-validation run, not a discovery validation.
- Public data checks are lightweight and summarize target-level evidence rather than publishing raw logs.
- Some targets reuse public source artifacts with different temporal tasks; target-level load counts are task loads, not claims of unique datasets.
- The validation cannot substitute for independent outside-party review.
- Classification labels remain sensitive to threshold choices, target filtering, replay caveats, and baseline definitions.
