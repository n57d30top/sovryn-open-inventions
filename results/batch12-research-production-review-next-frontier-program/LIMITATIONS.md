# Limitations

- This is a decision-grade review, not a new external target execution.
- Scores are based on public Batch 5-11 artifacts and should be treated as structured judgment, not statistical estimates.
- Execution-depth counts are lower bounds from public artifacts.
- The selected program can still fail if source protocols are ambiguous or target files cannot be loaded.
- The review intentionally narrows weak tools and domains instead of preserving prior optimism.
