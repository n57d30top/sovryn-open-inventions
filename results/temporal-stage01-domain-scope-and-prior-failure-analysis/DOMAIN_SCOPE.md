# Domain Scope

Temporal Evaluation Fragility means a claim about model or method quality changes materially when a temporal target is evaluated by chronological, rolling, blocked, horizon, random, or shuffled-time protocols.

Excluded: broad scientific discovery, package metadata residuals, sparse-table residuals, private data, medical/wet-lab/security-offensive targets, and any reuse of NEW-CAND-01 as a promoted candidate.
