# Major Limitations

1. Strong baselines explain many residuals.
2. Package popularity/proxy effects remain hard to separate.
3. Sparse-table missingness explains several table residuals.
4. Metadata count baselines dominate simple package descriptors.
5. Some replay checks preserve receipt but not residual strength.
6. Counterexamples are common enough to prevent validation-ready status.
7. Mutation unexpected results reduce confidence.
8. Controls do not produce false support but also reduce effect size.
9. Scope is narrower than the candidate name suggests.
10. No outside-party review or uptake is available or claimed.
