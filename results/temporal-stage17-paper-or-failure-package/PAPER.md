# Temporal Evaluation Fragility Deep Discovery Package

## Abstract
A domain-specific temporal evaluation study tested whether random split, temporal split, shuffled-time, horizon, and replay checks reveal a robust temporal principle. The result is a partial temporal signal, not a validation-ready candidate. Random splits can look easier on some high-autocorrelation or horizon-sensitive public series, but baselines, seasonality, counterexamples, and replay caveats narrow the claim.
