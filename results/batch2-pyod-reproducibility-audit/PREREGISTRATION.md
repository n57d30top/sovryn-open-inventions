# Preregistration

Research question: Which public repository artifacts are available for reproducing PyOD behavior without claiming a full package execution?

Metrics fixed before evaluation:
- reproducibility checklist score
- README presence
- dependency manifest presence
- examples/docs/tests presence
- lock-file presence

Kill criteria:
- repository inaccessible
- no dependency metadata
- claims test execution without running tests

Publication criteria: publish only if the result includes external target binding, concrete metrics, failures/losses, limitations, reproducibility instructions, and public hygiene pass.
