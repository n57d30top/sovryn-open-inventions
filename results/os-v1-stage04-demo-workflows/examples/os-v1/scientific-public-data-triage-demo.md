# Scientific Public Data Triage Demo

- Input: A public scientific data artifact with metadata and access notes.
- Command: `sovryn route plan --target scientific-public-data-triage-demo --json && sovryn route package --json`
- Expected package: public-data triage package with route, evidence status, and limitations
- Interpretation: Use the output to choose whether a deeper domain-specific audit is justified.
- Limitations: Does not make real-world scientific validation claims.
