# Tool Usefulness Demo

- Input: A public tool claim, README, and minimal usage example.
- Command: `sovryn route plan --target tool-usefulness-demo --json && sovryn route execute --target tool-usefulness-demo --json`
- Expected package: tool usefulness brief with static scan, route decision, and caveats
- Interpretation: Use the output to identify whether the tool has enough public evidence for a quick usefulness read.
- Limitations: Does not certify production fitness.
