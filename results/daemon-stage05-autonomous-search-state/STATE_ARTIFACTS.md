# State Artifacts

Required internal artifacts are created under `.sovryn/discovery-daemon/` by daemon init/status/run commands:

- `state.json`
- `candidate-identity-ledger.json`
- `graveyard.json`
- `search-cycles/`
- `fund-gate-results.json`
- `checkpoints/`
- `DAEMON_REPORT.md`
- `LIMITATIONS.md`

`FUND_FOUND.md` is created only when a Fund exists.
