# Validation Protocol

The validation freezes classifier version ca9fbfc, then maps deterministic blind slots to selected public temporal targets after the freeze. The run evaluates whether the instrument is only a conservative filter or can also identify robust temporal-fragility candidates.

## Inclusion
- Safe public temporal data artifacts
- Forecasting, temporal classification or sequence, anomaly, low-risk control, and mixed hard targets
- Targets with inspectable public source URLs
- Targets suitable for summarized retrieval receipts

## Exclusion
- Private data
- Unsafe domains
- Targets requiring raw log publication
- Targets whose claim requires outside-party review
- Targets from prior main evidence except explicit controls
