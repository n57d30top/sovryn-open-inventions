# Domain Selection

Selected domain: tabular and image-derived classification benchmark validation.

Why selected:

- Batch 9 showed that a public classification benchmark can support deep schema, metric, baseline, extension, and replay work.
- Batch 10 narrowed the custom tool claims and required every tool to prove value beyond pandas/sklearn baselines.
- This domain has public datasets, measurable baselines, safe computational scope, and enough repeated structure for multi-target comparison.

Rejected or deferred domains:

- Repository test reproduction was deferred because Batch 11 needed comparable dataset-and-metric runs rather than repo-specific dependency work.
- Source/claim extraction was deferred because the active toolchain under review is strongest for schema, metric, and replay packaging.
- Time-series anomaly baselines were deferred because cross-target split protocols are less comparable for this week's tool-value question.
