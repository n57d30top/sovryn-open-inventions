# Batch 11 Research Program Report

Batch 11 executed a coherent multi-target research program in tabular and image-derived classification benchmark validation.

Completed targets: UCI Rice Cammeo/Osmancik, UCI Optical Recognition of Handwritten Digits, UCI Iris, UCI Wine Recognition.

What happened:

- Four public targets were loaded and executed.
- The Vehicle target was attempted and replaced with Iris after a real target issue.
- Both required custom tools were used on every completed target.
- Dummy, linear, and tree baselines were run on every completed target.
- Metric stress checks and per-class weakness analysis were run on every completed target.
- Eight class-weight extension attempts were run; seven were rejected or downgraded.
- Two fresh-seed replays and two network-off container target replays were completed.

Main conclusion:

The toolchain is useful for evidence discipline and anti-hype checks, not as a broad raw-discovery engine for these single-table targets.
