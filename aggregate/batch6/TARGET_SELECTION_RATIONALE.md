# Target Selection Rationale

Batch 6 selected three public-safe targets that could be executed quickly without building a new framework layer:

- batch6-scikit-learn-iris-reproduction-ladder: public scientific or ML repository with tests/examples.
- batch6-uci-concrete-baseline-reproduction-ladder: public benchmark dataset with measurable baseline.
- batch6-diamonds-data-quality-netoff-ladder: public data-quality target.

The selection intentionally favors execution depth over count. The targets cover an ML repository/package smoke path, a measurable public benchmark dataset, and a data-quality replay under network-off isolation.
